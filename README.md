# Challenge-1-Python

## Propósito del análisis

Este proyecto tiene como objetivo analizar el desempeño de cuatro tiendas para ayudar al señor Juan a tomar una decisión sobre cuál tienda vender e invertir ese capital en un nuevo negocio.

Para ello, se realizó un análisis exploratorio de datos considerando indicadores como:

- Ingreso total por tienda
- Ventas por categoría
- Valoración promedio de los clientes
- Productos más vendidos y menos vendidos
- Costo de envío promedio por tienda

A partir de estos resultados, se entrega una recomendación final basada en los datos.
